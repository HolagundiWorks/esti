## Who buys

Indian **AEC advisory** firms — architecture studios and engineering consultancies
(structural / MEP / civil), roughly **5–50 people**.

**Not for:** contractors, PMC portfolios, construction execution ERP, warehouses,
running-account bills, or tenders.

---

## Job to be done

Replace WhatsApp + email + Excel + generic PM + ad-hoc GST with one **office
operating record**: fees → delivery → revisions → site supervision → invoices →
portals.

**Wedge line:** *Operating system for AEC consultancies in India*.

---

## Why AORMS (not Excel / ArchiO / Monograph)

| Need | AORMS |
|---|---|
| India money | Paise, COA fees, GST/TDS, FY April |
| Site supervision (architect-side) | Drawings, transmittals, snags, instructions, progress |
| Dual AI | **ESTI** (firm record) + **EOMS** (codes / knowledge bank) |
| Two practices, one spine | **AORMS-Studio** + **AORMS-Consultancy** |
| External proof | Client / consultant / contractor / site portals |
| Switching cost | Web-only · one Standard licence |
