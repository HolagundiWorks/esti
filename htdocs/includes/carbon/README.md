# ESTI Carbon UI Assets

This directory vendors the browser assets used by the ESTI Carbon product UI.

Packages:

- `@carbon/styles` `1.108.0`, Apache-2.0
- `@carbon/icons` `11.82.0`, Apache-2.0
- `@ibm/plex` `6.4.1`, OFL-1.1, stored under `htdocs/includes/fonts/ibm-plex`

Only distributable CSS, selected SVG icons, IBM Plex Sans, IBM Plex Sans
Devanagari, package metadata, and license files are committed.
